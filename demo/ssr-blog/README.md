# SSR Blog Demo

Demonstrates a `[slug]` blog route in `hydrated` mode with server-side data loading, route-level `hydrate.ts`, bootstrap payload rendering, compiler-loaded filesystem routes, automatic app handoff through `hydrateApp({ routes })`, and route components authored against `van-stack/render`.

For the quickest evaluator flow, start with `bun run start` and `demo/showcase`. This folder stays as the focused reference for the SSR blog route after the six-mode showcase.
